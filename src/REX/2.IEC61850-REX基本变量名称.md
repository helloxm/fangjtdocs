## 六、硬件板卡 I/O 类逻辑节点

这些逻辑节点用于映射继电器各插槽（Slot）的硬件板卡输入输出信号。命名规则如下：

- **X** = 扩展/硬件前缀
- **数字** = 板卡插槽号（90=COM/X000, 100=PSM/X100, 105=BIO/X105, 110=BIO/X110, 115=X115, 120=AIM/X120, 130=X130）
- **GGIO** = 通用I/O（Generic GGIO）
- **前缀字母含义**：B=高速输出(Boost/Binary high-speed), A=扩展模拟量(Advanced AIM), R=RTD温度, S=SIM(传感器接口), ARG=AIM+RTD组合, HB=心跳/高速二进制, BRG=BIO+RTD组合

| IEC 61850 名称 | 中文功能描述 | 对应硬件 |
|---|---|---|
| XGGIO90 | COM通信板卡(X000) I/O | COM (X000) 板卡 -- 含LDM功能 |
| XGGIO100 | 电源板卡(X100/PSM) I/O | PSM (X100) -- 电源模块上的二进制输入/输出 |
| XGGIO105 | BIO板卡(X105) I/O | BIO (X105) -- 二进制输入/输出板卡（REX615扩展槽位） |
| XRGGIO105 | BIO+RTD板卡(X105) RTD输入 | X105 槽位 -- RTD温度输入部分 |
| XHBGGIO105 | BIO板卡(X105) 高速二进制输出 | X105 槽位 -- 高速二进制输出（心跳/快速响应） |
| XGGIO110 | BIO板卡(X110) I/O | BIO (X110) -- 标准二进制输入/输出 |
| XBGGIO110 | BIO板卡(X110) 高速输出 | BIO (X110) -- 高速信号输出 |
| XRGGIO110 | BIO板卡(X110) RTD输入 | X110 槽位 -- RTD温度输入部分（如有） |
| XAGGIO115 | 扩展AIM板卡(X115) 模拟量I/O | X115 槽位 -- 扩展模拟量输入（电流/电压互感器） |
| XBGGIO115 | 扩展BIO板卡(X115) 高速输出 | X115 槽位 -- 高速二进制输出 |
| XGGIO120 | AIM板卡(X120) I/O -- 标准（4CT+Io/Uo） | AIM (X120) -- 4CT配置的模拟量输入 |
| XAGGIO120 | AIM板卡(X120) I/O -- 扩展（7CT/4CT+3VT） | AIM2 (X120) -- 7CT或4CT+3VT配置 |
| XGGIO130 | BIO板卡(X130) I/O | BIO (X130) -- 二进制输入/输出板卡 |
| XAGGIO130 | AIM板卡(X130) 模拟量I/O（5VT） | AIM (X130) -- 5路电压互感器输入 |
| XRGGIO130 | RTD板卡(X130) 温度输入 | RTD (X130) -- 6路RTD + 2路mA输入 |
| XSGGIO130 | SIM板卡(X130) 传感器I/O | SIM (X130) -- 3Is+3Us带灵敏Io传感器接口 |
| XARGGIO130 | AIM+RTD板卡(X130) 组合I/O | AIM+RTD (X130) -- 5VT + 2RTD + 1mA组合 |
| XBRGGIO130 | BIO+RTD板卡(X130) 组合I/O | X130 槽位 -- BIO+RTD组合板卡 |
